# Simple Calculator

This Java program implements a simple calculator using Java Swing. The calculator allows users to perform basic arithmetic operations such as addition, subtraction, multiplication, and division.

## Features

- User-friendly interface with buttons for digits (0-9) and arithmetic operations (+, -, *, /)
- Text field to display the result of calculations
- Proper arrangement of buttons and text field for easy interaction
- Exception handling to prevent errors such as division by zero

## How to Use

1. Run the program by compiling the Java file:

    ```bash
    javac Calculator.java
    java Calculator
    ```

2. Enter numbers and perform operations using the provided buttons.
3. The result of calculations will be displayed in the text field.
4. Click the "Clear" button to clear the text field.
5. Click the "Delete" button to delete the last entered character.
6. Note: Division by zero will be handled gracefully, and an error message will be displayed.


## Requirements

- Java Development Kit (JDK) installed on your system
- Any Java IDE or text editor for code editing (optional)
