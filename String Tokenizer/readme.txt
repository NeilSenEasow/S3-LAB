# Integer Sum Calculator

This Java program reads a line of integers from the user, displays each integer, and calculates the sum of all the integers using the `StringTokenizer` class from the `java.util` package.

## How it Works

1. The program prompts the user to enter a string containing integers separated by spaces.
2. It then uses the `StringTokenizer` class to tokenize the input string based on space delimiter.
3. It iterates through the tokens and converts each token to an integer using `Integer.parseInt()`.
4. The program calculates the sum of all integers and displays each integer along with the total sum.

## Usage

1. Compile the Java program using a Java compiler:

    ```shell
    javac StringTokenizer1.java
    ```

2. Run the compiled program:

    ```shell
    java StringTokenizer1
    ```
